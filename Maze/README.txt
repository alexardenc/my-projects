Rules of the game:
The program will solve the maze from mazedata.txt
1. Start at row 0, column 1
2. Move along the grid adding each number you pass to total sum
3. Try to get to row 1, column 9 keeping the total sum 0 or less

How to play:
1. Download file folder 
2. Run sumMaze.py
3. Press Enter to see next move
4. Program will find the solution
